# Figure Guide

Recommended figures for professor-facing updates:

- `goal1_finding_summary_board.png`: One-page visual summary of the main Goal 1 findings.
- `process_taxonomy_flow_story.png`: Flow-style view of creative conversation process taxonomy.
- `creative_run_length_survival.png`: Survival-style line chart showing how quickly creative runs end.
- `manual_transition_story_ci.png`: Human-calibrated wide/local next-turn transitions with bootstrap 95% CI.
- `pca_user_process_clusters.png`: PCA projection of conversation-level features, colored by the three user-process groups.
- `topic_process_bubble_map.png`: Topic-family bubble chart linking strict brainstorming rate, creative-state switches, conversation count, and mean turns.
- `sample_construction_workflow_cn.png`: Chinese workflow showing how the analysis moves from raw conversations to turn-level creative labels, creative conversations, and human-calibrated wide/local process analysis.
- `human_calibrated_multiturn_sankey_cn.png`: Chinese Sankey-style flow showing what happens after genuine multiple-option AI responses across later user turns.
- `manual_calibrated_next_turn_transitions.png`: Human-calibrated next-turn transition rates with bootstrap 95% confidence intervals.

- `process_taxonomy_distribution.png`: Distribution of creative conversation process types.
