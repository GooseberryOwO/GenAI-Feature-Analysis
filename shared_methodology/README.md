# Shared Methodology

此目录包含两个 research goals 共用的定义、四分类 turn-level classifier、方法说明和隐私边界。

Classifier 区分：

1. strict brainstorming
2. single creative generation
3. refinement / selection
4. non-creative task

它不直接输出 semantic wide/local transition；wide/local 需要结合 genuine multiple-option context 和人工校准。
